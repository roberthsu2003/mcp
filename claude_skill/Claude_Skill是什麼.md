# Claude Skill 是什麼?

A simple, open format for giving agents new capabilities and expertise.

Agent Skills are folders of instructions, scripts, and resources that agents can discover and use to do things more accurately and efficiently.

簡單來說，Claude Skill 是一種簡單的開放格式，用於給 agent 新的能力和專業知識。

Agent Skills 是一個包含指令、腳本和資源的文件夾，agent 可以發現和使用這些資源來更準確和高效地完成任務。

Why Agent Skills?
為什麼要使用 Agent Skills?

Agent are increasingly capable, but often don't have the context they need to do real work reliably.Skills solve this by giving agents access to procedural knowledge and company-, team-, and user-specific context they can load on demand.Agents with access to a set of skills can extend their capabilities based on the task they’re working on.

Agent 越來越有能力，但往往缺乏完成工作所需的上下文。Skills 解決了這個問題，透過給 agent 存取流程知識和公司、團隊、用戶特定的上下文，可以在需要時加載。

擁有存取一組 skills 的 agent 可以根據他們正在工作的任務來擴展他們的能力。

For skill authors: Build capabilities once and deploy them across multiple agent products.
For compatible agents: Support for skills lets end users give agents new capabilities out of the box.
For teams and enterprises: Capture organizational knowledge in portable, version-controlled packages.

對於 skill 作者：建立能力一次，並在多個 agent 產品中部署。
對於相容的 agent：支持 skills 讓最終用戶可以無需額外設定就能給 agent 新的能力。
對於團隊和企業：將組織知識打包成可攜帶、版本控制的套件。


What can Agent Skills enable?

Domain expertise: Package specialized knowledge into reusable instructions, from legal review processes to data analysis pipelines.
New capabilities: Give agents new capabilities (e.g. creating presentations, building MCP servers, analyzing datasets).
Repeatable workflows: Turn multi-step tasks into consistent and auditable workflows.
Interoperability: Reuse the same skill across different skills-compatible agent products.

領域專業知識：將專業知識打包成可重用的指令，從法律審查流程到資料分析管道。
新能力：給 agent 新的能力（例如：創建演示文稿、構建 MCP 伺服器、分析資料集）。
可重複的工作流程：將多步驟任務轉換為一致且可審計的工作流程。
互操作性：在不同的 skills-compatible agent 產品中重用相同的 skill。
