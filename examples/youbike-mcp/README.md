# YouBike MCP - 台北市 YouBike API 查詢系統

對應課程：**第二階段** - 外部 API 整合（5-8 節）

## 目標

- 串接台北市 YouBike Open API
- 將 API 封裝成 MCP Tool
- 提供自然語言查詢能力

## 功能

- 查詢某站點剩餘車輛數
- 查詢某區域站點資訊
- 查詢最近站點

## 學習重點

- RESTful API 設計思維
- API → MCP → UI 資料流
- 容器間網路設計
