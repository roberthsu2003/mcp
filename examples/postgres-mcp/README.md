# PostgreSQL MCP - 雲端資料庫整合

對應課程：**第三階段** - Render PostgreSQL 雲端資料庫整合（9-12 節）

- 連接 Render 雲端 PostgreSQL
- 讓 MCP Server 可查詢與寫入資料
- 設計「自然語言 → SQL → 回傳結果」流程

## 功能

- 資料表查詢
- 自然語言搜尋資料庫內容

## 學習重點

- MCP 作為資料搜尋層
